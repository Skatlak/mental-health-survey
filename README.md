# Опросник: Отношение к ментальному здоровью в ВУЗе

## Описание
Веб-приложение для сбора и анализа данных об отношении студентов к ментальному здоровью. 
Ключевые метрики: доступность помощи, уровень стигмы, востребованные инициативы.

## Технологии
- Python, Streamlit (Frontend/Backend)
- Firebase Firestore (База данных)
- Pandas, Plotly (Аналитика и визуализация)

## Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Убедитесь, что файл `serviceAccountKey.json` находится в корневой папке.
3. Запустите: `streamlit run app.py`
4. Откройте в браузере: `http://localhost:8501`

## Выводы по данным
Опрос показал, что студенты регулярно сталкиваются со стрессом и тревогой, но больше половины из них боятся идти к психологу из-за страха осуждения. Именно поэтому самым важным запросом стали анонимные чаты поддержки, где можно получить помощь без огласки. При этом сами ВУЗы рассказывает о психологах лишь на среднем уровне, а студенты в основном ищут информацию в соцсетях. По итогу, чтобы оказать помощь студентам, университету нужно активнее общаться с ними через социальные сети и запустить удобные анонимные чаты, которые снимут страх перед чужим мнением.
