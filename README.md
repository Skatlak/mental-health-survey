# Опросник: Отношение к ментальному здоровью в ВУЗе

## Описание
Веб-приложение для сбора и анализа данных об отношении студентов к ментальному здоровью. 
Ключевые метрики: доступность помощи, уровень стигмы, востребованные инициативы.

## Технологии
- Python, Streamlit (Frontend/Backend)
- Firebase Firestore (База данных)
- Pandas, Plotly (Аналитика и визуализация)

## Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Убедитесь, что файл `serviceAccountKey.json` находится в корневой папке.
3. Запустите: `streamlit run app.py`
4. Откройте в браузере: `http://localhost:8501`

## Выводы по данным
Анализ показал, что более 50% студентов сталкивались с неодобрением общества при обращение за психологической помощью. 
Доступность информации в вузе о психологической помощи находится в среднех показателях, а стресс и трвеожность испытывают со средней переодичности.
Так же самые популярные источники для получения информации это соцсети, а самая необходимая инициатива по поддержке это анонимные чаты поддержки.
